# Retail Supply Chain & Consumer Health Trends Analysis

This project simulates and analyzes supply chain performance and sales trends, with a particular focus on consumer health products. It demonstrates the end-to-end journey of a **Data Analyst / Data Engineer**: raw data cleaning/manipulation, daily inventory/stockout simulation, database construction, SQL-based business analysis, and interactive dashboarding.

## 🚀 Data Pipeline Workflow

Follow these steps to run the pipeline sequentially:

1. **Data Prep & Trend Simulation**:
   ```bash
   python data_prep.py
   ```
   *What it does:* Reads `raw_data.csv/supply_chain_dataset1.csv`, classifies products as `Health` or `Standard`, doubles demand for `Health` items, and runs a chronological daily inventory simulation loop to capture resulting stock depletion and stockout events.

2. **Database Initialization**:
   ```bash
   python create_db.py
   ```
   *What it does:* Reads the modified CSV data, writes it into a table called `retail_data` inside `supply_chain.db`, and executes the SQL queries in `analytics_metrics.sql` to construct pre-aggregated views.

3. **Metrics Analysis (SQL)**:
   ```bash
   python analysis.py
   ```
   *What it does:* Connects to `supply_chain.db` and runs SQL queries to print critical KPI summaries (Revenue and Sales, Stockout Risks, Top Warehouse Bottlenecks) directly to the console.

4. **Data Visualization**:
   ```bash
   python visualize.py
   ```
   *What it does:* Queries the database and outputs a high-resolution 3-panel report showing sales volume, stockout risk rate, and supplier delivery consistency, saving the plot as `supply_chain_insights.png`.

5. **Interactive Dashboard**:
   ```bash
   python -m streamlit run dashboard.py
   ```
   *What it does:* Opens a web-based interactive analytics dashboard in your browser.

## 📊 Streamlit Dashboard Features

The dashboard includes five specialized analytics tabs:
* **📈 Executive Overview:** Quick metrics (Revenue, Units, Lead Times, Alerts) and regional sales graphs.
* **🛡️ Inventory Optimization:** Dynamic safety stock and reorder point (ROP) calculator using the mathematical formula:
  $$\text{Safety Stock} = Z \times \sqrt{\bar{L} \cdot \sigma_d^2 + \bar{d}^2 \cdot \sigma_L^2}$$
  Allows users to adjust target service level percentages (90%, 95%, 99%) to get restocking recommendations.
* **🔮 Demand Forecasting:** Projections for the next 30 days of sales using a linear trend model with day-of-week seasonal indexing.
* **🤝 Supplier Risk Profile:** Scores suppliers out of 100 based on delivery lead times and timing consistency (standard deviation).
* **💻 Interactive SQL Sandbox:** Allows users to query the `retail_data` table directly inside the browser.
